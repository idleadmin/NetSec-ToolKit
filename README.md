# 🛡️ NetSec-ToolKit

A comprehensive repository of **real-world, practical scripts** designed for network security professionals and system administrators. This toolkit streamlines operations across multiple vendors and platforms, focusing on automation, security hardening, and efficiency.

## 📂 Featured Solutions

### 🛡️ Microsoft Ecosystem
- **Windows Client**: PowerShell scripts for disabling cached logons, autoplay, and restricting null sessions.
- **Windows Server**: Security hardening like disabling SMBv1.
- **Azure**: Cloud infrastructure management scripts.

### 🦅 CrowdStrike
- **IOC Management**: Automated tools for adding Indicators of Compromise.

### 🧱 Fortigate Firewall
- **Mass Execution**: Guides and scripts to run commands across multiple hosts simultaneously.

### 📧 Google Cloud / Workspace
- **AppScript**: Productivity tools like Gmail Auto-Categorizer.
